Problem Set - 3

Problem 3.1.Caesar Cipher
  • A Caesar cipher is a simple and easily crackable encryption method. The approach is to replace each letter of the secret message with a different letter of the alphabet which is ‘n’ positions further in the alphabet.
  • Eg.if n=3 “This is a cipher” becomes “Wklv lv d flskhu”
  • Since each letter in the message can directly be translated to another letter, you can use frequency of letters to decipher the code.
  • A more straightforward approach to crack a caesar cipher would be to try all values of n(1-25) and see what results.
  • How would you crack “Cqn zdrlt kaxfw oxg sdvyb xena cqn ujih mxp ” ?
  • What is n in this case?
  • Note: If letters exceed ‘z’ they are made to rotate back to ‘a’. Each word in the original message is a valid word in English.
Problem 3.2. N factors
  • Given a number N how would you find the smallest integer which has at least N factors (1 and the number included)?
  • i)N=36
  • ii)N=72
  • Hint: Use your computer’s help.
